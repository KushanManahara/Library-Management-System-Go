# üìö Library Management System in Go

Library Management System in Go is a web application built using the Gin framework to manage books in a library setting. It allows librarians and administrators to perform CRUD operations (Create, Read, Update, Delete) on a collection of books.

## üöÄ Features

- **Manage Books:** Add, edit, and delete books with title, author, and year information.
- **User-Friendly Interface:** HTML templates provide a responsive and intuitive user interface.
- **Static File Handling:** Serve static assets like CSS files for enhanced styling.

## üíª Installation

To run the Library Management System locally, follow these steps:

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/YourUsername/Library-Management-System-Go.git

2. Navigate to the project directory:


    ```bash
    cd Library-Management-System-Go
    ```

3. Install dependencies:

    ```bash
    go mod tidy
    ```

4. Build and run the application:

    ```bash
    go run main.go
    ```

5. Open your web browser and go to `http://localhost:8080` to view the application.






## üöÄ Usage

Navigate through the application to manage books:

- `/` : View all books.

- `/books/new` : Add a new book.

- `/books/edit/:id` : Edit a specific book.

- `/books/delete/:id` : Delete a specific book.
